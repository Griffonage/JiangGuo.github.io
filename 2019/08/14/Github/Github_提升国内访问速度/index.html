Github_提升国内访问速度
发表于2019-08-14|更新于2020-09-18|Github
|字数总计:194|阅读时长:1分钟

在国内访问Github时会出现速度太慢问题,我的Picgo上传到Github图床也会出现无法预览情况

除了梯子,可以通过修改host,达到加速的效果

访问:https://www.ipaddress.com/

分别查询

  • github.com

  • github.global.ssl.fastly.net

得到两个域名的IPV4地址,然后修改电脑目录下:C:\Windows\System32\drivers\etc的hosts文件内容

按照上面的格式填好,再刷新下电脑的host缓存,windows键+R输入cmd,调出命令窗口输入ipconfig /flushdns即可完成

注意:对于自建图床再Github上的,需要再添加一条raw.githubusercontent.com的解析地址,同上,因为Github对资源的存放位置的不一样
