语言概述

发展简历

由Sun公司在1995年发布了Java,在1998年发布了JDK1.2版本,包括有JSP/Servlet、EJB等规范,并将Java分为J2EE、J2SE、J2ME三个版本

  • J2SE:Java核心和基础

  • J2ME:主要用于控制移动设备和信息家电等有限存储设备

  • J2EE:应用广泛,提供了企业级应用开发方案
